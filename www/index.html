About Trevorese Typing Assistant

This tool helps you type in Trevorese, a constructed language. It provides real-time translation between glosses and surface forms, and offers suggestions for unknown words.

How to use:

  • Enter text in the "input (gloss)" box in the "gloss to surface" tab.
  • Trevorese glosses are separated by spaces.
  • Hyphens are used within a single Trevorese word.
  • See the output appear in real time in the "surface" box.
  • If you enter English, the assistant will check the dictionary for possible glosses.
